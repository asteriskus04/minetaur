Добро пожаловать в Mine'тавр!

 Mine'тавр - программа, разработанная с целью помочь сетевому администратору в обнаружении несанционированного использования вычислительных мощностей оконечных устройств в локальной сети.

 Майнинг - это деятельность по созданию новых структур для обеспечения функционирования криптовалютных платформ. За создание очередной структурной единицы обычно предусмотрено вознаграждение за счёт новых единиц криптовалюты и/или комиссионных сборов. Обычно майнинг сводится к серии вычислений с перебором параметров для нахождения хеша с заданными свойствами.

Добро пожаловать в Mine'тавр!

Mine'тавр - это Web-приложение по выявлению инцидентов несанкционированного использования вычислительных мощностей c помощью нейронной сети при анализе ключевых элементов инфраструктуры распределенных компьютерных сетей.

Система «Mine'тавр» рекомендована к использованию в случаях, при проверке компьютерных сетей и их элементов на предмет использования вычислительных мощностей для добычи криптовалюты.

Нейронная сеть позволяет на основе атрибутов, полученных из трафика, оценить риск несанкционированного использования вычислительных мощностей компьютерных сетей и с высокой точностью обнаружить заражённое устройство!
